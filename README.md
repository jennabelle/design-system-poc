# design-system-poc
proof-of-concept for universal design system

To setup:
```
git clone
npm install
```

To run Storybook:

`npm run storybook`

Then go to:

`http://localhost:6006/`

To create new package within monorepo, go inside the packages folder:

```
npx create-react-app [name of package]
cd [name of package]
npm install
npm start
```
